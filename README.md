## Game Default Key Bindings
| Key                 | Action                       |
|---------------------|------------------------------|
| `W,A,S,D`           | Movement                     |
| `SHIFT`             | Dash                         |
| `Left Click, Space` | Shoot                        |
| `B`                 | Open inventory               |
| `ESC`               | Pause game                   |
| `F4`                | Debug mode                   |
| `F11`               | Switch fullscreen / windowed |
